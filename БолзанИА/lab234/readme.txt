Что делают радиокнопки Live и Extract?
---------------------------------------
Отвечают за чтение данных из источников. Live метод нужен для чтения быстроменяющихся в реальном времени данных,
соответственно, требует больших затрат ресурсов. Extract наоборот подходит для случаев,
когда необходимо провести анализ накопленных данных и требует меньших затрат.


Как просмотреть на графике средние значения по показателю?
-----------------------------------------------------------
Для этоо необходимо на дропдаунте рассматриваемого значения поменять Sum установленный по умолчанию, показывающий
например, общее число продаж на Average.


Что такое Show me?
-------------------
Show me это подбор графиков. Эта функция сама предлагает наиболее удачный и удобный график на основе
выбранных пользователем критериев. Для этого не обходимо выбрать эту функцию, зажать ctrl и выбрать интересующие
критерии.



Что такое линия тренда?
-----------------------
Линии тренда — инструмент технического анализа, используемый для выявления тенденций изменения цен.
Она представляют собой геометрическое отображение средних значений анализируемых показателей, 
полученное с помощью какой-либо математической функции. 



Как пременить фильтр для всех рассматриваемой отчетности в режиме отчета?
--------------------------------------------------------------------------
Если один из листов содержал фильтр по, например, категориям, то можно выбрать окошко отвечающее за фильтрацию,
Apply to Worksheets -> All Using This Datasource. 
После этого листы будут отражает показатели одной категории.



Для чего нужны Истории в Tableau?
-----------------------------------
Истории позволяют создать полностью интерактивный отчет с возможностью добавления описания.
