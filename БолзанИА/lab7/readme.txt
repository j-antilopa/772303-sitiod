Какие существуют способы для нахождения остатков?
------------------------------------------------------
 1) через формулу с помощью переменных коэффициентов
fit$coefficients[[2]]*year+fit$coefficients[[1]]
 2) С помощью команды residuals
 3) с помощью аттрибута переменной: fit$residuals

Что такое пояснительная переменная и переменная отклика?
----------------------------------------------------------
Пояснительная (экзогенная) это изменяющаяся переменная которая влияет на изменение ставки во времени.
Переменная отклика - эндогенная переменная значения которой непосредственно зависит от пояснительной с
влиянием отклонений.
