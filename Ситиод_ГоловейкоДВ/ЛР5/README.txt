1.Для чего используются инструменты языка R?
Для более удобной работы с последовательностями чисел, матрицами, векторами и операциями над ними с помощью встроенных инструментов.

2.За что отвечает функция prod?
Функция prod() отвечает за возвращение произведения всех чисел последовательности.

3.Для чего используется библиотека pandas?
pandas - библиотека для работы с данными, представленными в форматах csv, sql и т.п., можно сказать, что pandas необходим для ETL и основывается на numpy
