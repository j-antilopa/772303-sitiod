1.Для чего используется программа Tableau? 
Tableau — серия продуктов визуализации и обработки данных, которая используется для создания бизнес-аналитики и визуальной отчетности.

2.Зачем использовать визуализацию данных?
Визуальная информация лучше воспринимается и позволяет быстро и эффективно донести до зрителя собственные мысли и идеи. Физиологически, восприятие визуальной информации является основной для человека.

3.Как визуализировать данные в программе в программе Tableau?
Для того чтобы визуализировать данные, необходимо переместить нужные поля в соответствующие разделы колонок и строк.

4.Можно ли улучшить цветовое представление данных?
Да, можно. Для этого необходимо использовать «Colors».
