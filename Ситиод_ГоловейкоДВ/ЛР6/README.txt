1.С помощью какой функции можно визуализировать другие функции при помощи встроенных инструментов языка R?
За это отвечает функция plot().

2.Каким способом можно решить матричные уравнения с помощью встроенных инструментов языка R?
Можно воспользоваться функцией solve(), принимающая коэффициенты уравнения.

3.Что такое срезы в python/numpy/pandas?
Срез (slice) — извлечение из данной строки одного символа или некоторого фрагмента (подстроки).
