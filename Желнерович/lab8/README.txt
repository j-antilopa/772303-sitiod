1. Для каких приложений оптимизированы базы данных NoSQL?
   - Базы данных таких типов оптимизированы для приложений, которые работают с большим объемом данных, нуждаются в низкой задержке и гибких моделях данных, например, мобильные, игровые, интернет-приложения, когда требуются гибкие масштабируемые БД с высокой производительностью и широкими функциональными возможностями.Xnj

2. Что такое коллеция?
   - Коллекция - это эквивалент простой таблицы в реляционной БД. 

3. Что такое документ?
   - Документ - это набор пар "ключ-значение".

4. Какой метод используется для структурированного вывода данных коллекции?
   - pretty().
