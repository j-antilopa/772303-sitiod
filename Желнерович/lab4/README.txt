1. Что такое Dashboards и для чего они нужны?
   - Dashboards - документы с представленными графиками и отчетами. Они позволяют переводить большое количество данных на страницу с краткими результатами (описанием).

2. Как настроить параметры визуализации Dashboards
   - С помощью кнопки Device Preview, сверху над представлением панели мониторинга добавится панель визуализации.

3. Какие критерии для хорошего представления Dashboards?
   -Графики должны быть визуально приятными (в одном цветовом тоне, с наглядно отображенными на них данными и т.д.), тип визуализации должен быть подходящим по смыслу задачи и должно небольшое количество текста и больше визуализации.

4. Чем истории отличаются от дашбордов?
   - По сути, истории  - это несколько рабочих листов, которые также могут включать в себя дашборды, представленные в виде некоторой последовательности изложения данных.
