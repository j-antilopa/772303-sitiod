1. Что такое Tableau?
   - Tableau – это программа для визуализации данных. Данное ПО позволяет разработчикам визуально представлять данные и информацию, которую они собирают.

2. Что такое визуализация данных?
   - Визуализация данных - способ представления собранных данных в визуально привлекательной форме. Это обычно включает строки, таблицы и все, что с этим связано.

3. В каких ситуациях используется «Join» и в каких «Blending»? 		
   - Обе команды используются для подключения первичного источника данных ко вторичному. В ситуациях, когда данные находятся в одном месте, потребуется Join. Если же данные разбросаны по нескольким разным местам, необходимо использовать Blending.

4. Сколько таблиц можно объединить в Tableau? 
   - 32 таблицы
