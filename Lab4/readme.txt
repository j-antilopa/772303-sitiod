1. Для чего нужен отчет?
Простроив графики и проведя необходимые анализы, необходимо отправить всё это 
для анализа коллегам. Для этого и нужен отчет - для обобщения сделанной работы.

2. Что такое история в Tableau и для чего нужна?
Истории позволяют создавать интерактивные отчеты с возможностью оставить
комментарии, добавить опиасние к деталям.
