1. R - язык программирования для статистической обработки данных и работы с графикой.

2. Data Mining - интеллектуальный анализ данных, в целом совокупность методов при решении конкретных задач анализа данных с целью выявления скрытых закономерностей.

3.Линейная регрессия - модель зависимости переменной x от других переменных (функция зависимости линейна).

4.Корреляция - взаимозависимость двух или более сдучайных велечин.

5.Команда lm - команда для выполнения регрессии наименьших квадратов(в качестве атрибутов используются вектор, содержащий переменную отклика, и вектор, содержащий объясняющую переменную.
